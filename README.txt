1. PROJECT DESCRIPTION:
This application is a graphical upgrade to the traditional 
text-based Tic-Tac-Toe. It uses Python's Tkinter library to 
create an interactive 3x3 grid. The game is designed for two 
players, featuring automated win detection and turn management.

2. TECHNICAL SPECIFICATIONS:
- Language: Python 3.x
- Library: Tkinter (Built-in)
- Paradigm: Object-Oriented Programming (OOP)
- Input Method: Mouse-click events (Event-Driven)

3. KEY FEATURES:
- Clickable 3x3 Grid: No keyboard input required.
- Turn Tracker: Real-time status label showing player turns.
- Win Logic: Checks 8 patterns (Rows, Columns, Diagonals).
- Color Feedback: X is displayed in BLUE; O is displayed in RED.
- Game Reset: One-click button to clear the board and restart.

4. HOW TO RUN:
- Open the 'TICTAKTO.PY' folder in VS Code.
- Ensure 'main.py' is open.
- Run the script using the 'Play' button or press F5.
- A game window will pop up. Click any square to begin.
